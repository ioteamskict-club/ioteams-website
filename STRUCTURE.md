# IoTeams Website — Folder Structure

## Root
- `index.html`         → Homepage
- `pages/`             → All inner pages (events, gallery, blog, etc.)
- `forms/`             → Backend form handlers (contact.php, apply.php)
- `assets/`            → All custom project files
- `vendor/`            → All third-party libraries (never edit these)

## assets/
- `css/main.css`       → Main custom stylesheet (combined Web1 + Web2 styles)
- `js/main.js`         → Main custom JavaScript
- `img/hero/`          → Hero/banner images
- `img/gallery/`       → Gallery page images
- `img/blog/`          → Blog post images & author photos
- `img/team/`          → Team/member profile photos
- `img/events/`        → Event photos
- `img/icons/`         → Logo, favicon, custom icons

## vendor/
- `bootstrap/css/`     → Bootstrap 5.3.3 CSS files
- `bootstrap/js/`      → Bootstrap 5.3.3 JS bundle
- `aos/`               → AOS (Animate On Scroll) library
- `swiper/`            → Swiper.js (carousel/slider)
- `glightbox/`         → GLightbox (image lightbox for gallery)
- `bootstrap-icons/`   → Bootstrap Icons font files
- `php-email-form/`    → PHP Email Form library (pro - contact form)

## forms/
- `contact.php`        → Contact form handler
- `apply.php`          → Apply/join form handler

## pages/
- `our-story.html`     → About / team page
- `events.html`        → Events listing
- `gallery.html`       → Photo gallery
- `blog.html`          → Blog listing
- `blog-details.html`  → Single blog post
- `programmes.html`    → IoT programmes/tracks
- `apply.html`         → Join IoTeams form
- `contact.html`       → Contact page
