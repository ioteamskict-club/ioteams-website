◆ Track 01

Hardware &
Embedded Systems

Every IoT system starts with hardware. In this track you'll go from blinking an LED to building fully functional embedded systems that read sensors, control actuators, and communicate wirelessly.

No prior experience needed. We start from scratch and build up to advanced topics like RTOS, power management, and custom PCB design.

What You'll Learn:
  • Arduino & ESP32 programming in C/C++
  • Digital & analog I/O, PWM, interrupts
  • Sensor integration — temperature, humidity, motion, ultrasonic
  • Wireless protocols — WiFi, Bluetooth, LoRa
  • Real-Time Operating Systems (FreeRTOS)
  • Power management & battery optimisation
  • PCB schematic & layout with KiCad
M01
Introduction to Microcontrollers

What is a microcontroller? Setting up Arduino IDE, first sketch, digital I/O.

2 Sessions Beginner
M02
Sensors & Actuators

Reading sensors (DHT22, HC-SR04, PIR), controlling servos, relays, and motors.

3 Sessions Beginner
M03
ESP32 & Wireless Comms

WiFi, Bluetooth, ESP-NOW, deep sleep, and connecting to the internet for the first time.

3 Sessions Intermediate
M04
RTOS & Advanced Firmware

FreeRTOS tasks, queues, semaphores, and building production-grade embedded firmware.

4 Sessions Advanced
M05
PCB Design with KiCad

Schematic capture, PCB layout, design rules, and ordering your first custom PCB.

3 Sessions Advanced
M01
Networking Fundamentals for IoT

IP, TCP/UDP, HTTP, WebSockets — understanding the protocols that power IoT connectivity.

2 Sessions Beginner
M02
MQTT Protocol & Brokers

Publish/subscribe model, setting up Mosquitto broker, connecting ESP32 via MQTT.

3 Sessions Intermediate
M03
Cloud Platforms — AWS IoT & Firebase

Provisioning devices, Thing Shadows, IoT Rules, and real-time Firestore integration.

4 Sessions Intermediate
M04
Real-Time Dashboards with Node-RED

Visual programming for IoT flows — connecting sensors to live charts and alert systems.

2 Sessions Intermediate
M05
REST APIs & Serverless Functions

Building APIs for your devices using AWS Lambda and triggering actions from the cloud.

3 Sessions Advanced
◆ Track 02

Cloud &
Connectivity

Hardware alone isn't enough. In this track you'll learn how to get your devices talking to the internet, storing data in the cloud, and building dashboards that display live sensor readings.

Recommended to take alongside or after the Hardware track, though not strictly required. Python and basic networking knowledge is helpful.

What You'll Learn:
  • IoT networking protocols — MQTT, HTTP, WebSocket
  • Cloud platforms — AWS IoT Core, Firebase, Azure
  • Building real-time dashboards with Node-RED
  • REST API design and consumption
  • Serverless functions with AWS Lambda
  • Data storage — InfluxDB, DynamoDB, Firestore
  • OTA firmware updates and device management
◆ Track 03

AI &
Smart Systems

The most exciting frontier in IoT — making devices intelligent. This track covers Edge AI, TinyML, and computer vision, enabling you to run machine learning models directly on microcontrollers.

This is an advanced track. Basic Python and some hardware experience is recommended. We'll guide you from model training to deployment.

What You'll Learn:
  • Machine learning fundamentals for IoT
  • TensorFlow Lite & Edge Impulse
  • TinyML — running ML on Arduino Nano 33 BLE
  • Computer vision with OpenCV on Raspberry Pi
  • Anomaly detection in sensor data streams
  • Keyword spotting and gesture recognition
  • Optimising models for constrained devices
M01
ML Fundamentals for IoT Engineers

Classification, regression, neural networks — explained for hardware people, not data scientists.

2 Sessions Intermediate
M02
Edge Impulse & TinyML Workflow

Collect data, train a model, and deploy it to an Arduino in under 2 hours using Edge Impulse Studio.

3 Sessions Intermediate
M03
Computer Vision on Raspberry Pi

Object detection, face recognition, and image classification using OpenCV and TensorFlow Lite.

4 Sessions Advanced
M04
Anomaly Detection in Sensor Streams

Detecting equipment failures and unusual patterns in real-time sensor data using lightweight models.

3 Sessions Advanced
M05
Capstone — Build an AI-Powered Device

End-to-end project: design, train, deploy, and demo a real AI IoT system of your own choice.

4 Sessions Advanced
